human-web

Tiny 3D characters that walk around your website. One script tag. No dependencies. They walk, run, sit, and even camouflage into the background.

<script src="https://cdn.jsdelivr.net/gh/ShotaNagafuchi/human-web@main/dist/human-web.js"></script>
GitHub

One Line Install

Add a single <script> tag to your HTML. Characters appear automatically.

<script src="https://cdn.jsdelivr.net/gh/ShotaNagafuchi/human-web@main/dist/human-web.js"></script>

JavaScript API

Control characters programmatically.

HumanWeb.triggerAll('chameleon'); // hide in background
HumanWeb.triggerAll('run');       // sprint!
HumanWeb.addCharacter();          // add more

Dark Background

Chameleon mode samples this dark area and paints it onto the character's body.

Light Background

Works on any background color. The characters adapt to whatever is behind them.

Features

Walk, run, idle, sit, and chameleon behaviors. Mouse head tracking. Click to reveal camouflaged characters with an ink splash effect. Bone animation editor included.
